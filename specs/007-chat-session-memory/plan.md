# Implementation Plan: Chat Session Memory

**Branch**: `007-chat-session-memory` | **Date**: 2026-02-23 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/007-chat-session-memory/spec.md`

## Summary

Add session-scoped conversation history to the RAG chat. The Angular `messages` signal already holds the full conversation; this feature forwards the last N turns to the backend with each query. The backend includes them in the LLM prompt when present. No DB changes, no new dependencies, full backward compatibility.

## Technical Context

**Language/Version**: Python 3.12 (backend), TypeScript 5.9 / Angular 21 (frontend)
**Primary Dependencies**: FastAPI, LangChain 0.3, Pydantic v2 (backend); Angular 21, PrimeNG v21 (frontend)
**Storage**: No new storage — history is ephemeral (client-side `messages` signal)
**Testing**: pytest + ruff (backend), Karma/Jasmine + ESLint (frontend)
**Target Platform**: macOS local dev, single-user
**Performance Goals**: Query latency unchanged (under 10s p95 on local stack)
**Constraints**: No DB schema changes, no new pip/npm packages, backward-compatible API

## Phase 0: Research

See [research.md](./research.md) — all decisions resolved:
- History format: labeled plain-text (`Utilisateur: / Assistant:`)
- Placement: before retrieved context in prompt
- Window: frontend sends last 6 turns (12 messages); backend accepts up to 10
- Guardrails: rejected messages not included (client-side: only successful messages in signal)
- No server-side state: client sends history per request

### Backend

**`rag/prompts.py`** — add alongside existing `RAG_PROMPT`:
```python
RAG_PROMPT_WITH_HISTORY = """...(same rules)...

Historique de la conversation :
{history}

Contexte :
{context}

Question : {question}

Réponse :"""
```

**`rag/pipeline.py`** — add helpers + update signatures:
```python
def _format_history(history: list) -> str:
    lines = []
    for entry in history:
        label = "Utilisateur" if entry.role == "user" else "Assistant"
        lines.append(f"{label} : {entry.content}")
    return "\n".join(lines)

def _build_prompt(context: str, question: str, history: list) -> str:
    if not history:
        return RAG_PROMPT.format(context=context, question=question)
    return RAG_PROMPT_WITH_HISTORY.format(
        history=_format_history(history),
        context=context,
        question=question,
    )
```

`query(question, history=[])` and `stream_query(question, history=[])` — call `_build_prompt()`.

**`api/v1/query.py`** — add:
```python
from typing import Literal

class HistoryEntry(BaseModel):
    role: Literal["user", "assistant"]
    content: str

class QueryRequest(BaseModel):
    question: str
    history: list[HistoryEntry] = []
```

Pass `history=request.history` to pipeline calls.
